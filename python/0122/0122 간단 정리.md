## list 내의 list들의 요소별로 묶인 list를 만들고, 배열에 집어넣기

```python
ex)
lst = [[1, 2], [3, 4], [5, 6]]
	>>  {
			'lst_1': [1, 3, 5],
			'lst_2': [2, 4, 6]
		}

def turn(temps):
	max_temp = []
	min_temp = []
	for tem in temps:
		max_temp += [temps[0]]
		min_temp.append(temps[1])
	result = {}
	result['max'] = max_temp
	result['min'] = min_temp
	
	return result
```
